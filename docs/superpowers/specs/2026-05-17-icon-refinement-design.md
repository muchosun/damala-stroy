# DAMALA STROY Landing — Icon Refinement Design

## Goal

Finalize the contact polish by replacing the hand-drawn SVGs with a consistent icon pack and slightly enlarging the mobile floating contact capsule.

## Agreed Direction

- Use **Bootstrap Icons** as the single icon source for:
  - Telegram
  - WhatsApp
  - Telephone
- Keep icons monochrome so they remain aligned with the premium visual language of the landing page.
- Preserve the existing desktop and mobile contact architecture.
- Increase the visual scale of the mobile floating contact capsule slightly so it reads more confidently at the bottom of the screen.

## Design Rules

1. Improve polish, not concept.
2. Keep the page visually restrained.
3. Make the mobile contact controls easier to perceive without turning them into a loud CTA.
4. Continue using Git-only rollback through milestone commits and tags.

## Expected Outcome

- Contact icons feel cleaner and more professional.
- Desktop header remains elegant.
- Mobile contact capsule becomes a little larger and easier to use.
- The landing page feels finished rather than reworked.
